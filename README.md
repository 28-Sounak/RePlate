# 🍽️ RePlate Backend

### Turning Surplus Into Support

Smart backend infrastructure for a real-time food redistribution platform connecting restaurants, NGOs, and volunteers to reduce food waste and fight hunger.

# 🚀 Features

- 🔐 JWT Authentication
- 🍛 Food Upload APIs
- 📍 NGO Matching System
- 🚚 Volunteer Coordination
- 🔔 Real-Time Notifications
- 📡 Live Tracking
- 📊 Analytics Module
- 🤖 AI Expansion Ready

# ⚙️ Installation

## 1️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 2️⃣ Run the Server

```bash
uvicorn app.main:app --reload
```

# 🔄 Core Workflow

```text
Restaurant uploads surplus food
        ↓
Nearby NGOs receive alerts
        ↓
Volunteer assigned
        ↓
Pickup & Delivery Tracking
        ↓
Food reaches communities
```
